# NTE（异环）真实游戏截图样本

把真实 NTE（异环 / Neverness to Everness）游戏内**包含自创文字**的截图放在这个目录下。

## 用途

1. **Phase 0**：和 TTF 渲染出的 glyph 做形状比对，验证字体是否为游戏同款
2. **Phase 5**：作为真实世界测试集，评估识别准确率

## 投放建议

- 文件格式：PNG / JPG 都可以
- 数量：Phase 0 验证只需 5-10 张；Phase 5 评测希望能凑到 20-50 张
- 多样性优先：不同光照、不同字号、不同背景、不同 UI 位置（对话框、招牌、信件等都要有）
- 命名随意，但建议带场景信息，如 `dialog_01.png` / `sign_02.jpg` / `letter_03.png`

## 不需要的

- 不要附带任何账号/个人信息可见的截图
- 不需要预先裁剪，原始分辨率截图最佳
